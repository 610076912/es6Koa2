
## 项目运行

```
1、node (8.0 及以上版本)
```

```
开发环境
npm run dev 


模拟生产环境
npm run satrt


代码编译
npm run build


访问: http://localhost:3889

```





# 目录结构

```
dist                  // 代码编译后的存放目录
logs                  // 日志
src                   // 源码
  config              // 一些配置文件
  controllers         // 
  middleware          // 中间件目录
  router              // 路由
babel-dsp.js        // 开发环境实时进行babel编译的入口文件
dsp.js              // 入口文件

```
